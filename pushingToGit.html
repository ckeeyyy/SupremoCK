These are the steps to push a github repository

First lets discuss what is git?
Git is a free and open source distributed control system designed to handle
everything from small to very large projects with speed and efficiency.

Here are the steps to push your code to your own repository. 

  
        1. Create a folder specifically for your project.  
             $ mkdir [yourFolderName]
            
            Output:

                $ mkdir webDevSupremo

        2. Change directory.
            $ cd [yourFolderName]

            Output:

                $ cd webDevSupremo

        3. Next is to open your VSCode.

        4. Create a new file in VSCode and then put it in the folder that you created (Refer to step #1).

            Output: 

                pushingToGit.html

        5. Input your code to your html file. 

        6. After coding, push and commit your file through Git Bash. 

        7. Make sure you're in the right directory
            $ pwd

            Output: 

                /c/Users/SMARTLAB PC 04/webDevSupremo
        
        8. Make sure you configure your credentials. 
            $ git config --global user@email "MY_NAME@example.com"

            Output:

                $ git config --global user.email "camilakim.supremo@hcdc.edu.ph"

        9. Add your remote repository in your local machine
            $ git remote add [localComputer] [url to your repository]

        10. Make sure you're in the correct branch
            $ git branch -M main

        11. Check your file using git status 
            $ git status

        12. Add all your code to the add phase
            $ git add -A 

        13. Then commit your code to the repository 
            $ git commit -M "Commite code"

        14. Push your code to the repository 
            $ git push -u (localComputer) (currentBranch)
